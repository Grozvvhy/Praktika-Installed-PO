## Описание репозитория

Данный репозиторий содержит все артефакты, подготовленные в ходе производственной практики на базе МБУК «Дворец культуры имени Горького» (ДК «Паруса»).  
В рамках практики была спроектирована и развёрнута защищённая информационная система, включающая контроллер домена Active Directory, файловый сервер, систему 1С:Предприятие, офисный пакет Microsoft Office, антивирусную защиту Kaspersky Endpoint Security, а также выполнена сборка открытого ПО (SQLite) из исходных кодов.

Здесь представлены:

- отчёт о практике (`.docx`, `.pdf`);
- дневник практики;
- аттестационный лист;
- скриншоты выполненных работ;

## Технологический стек
- **Серверная ОС:** Windows Server 2022 (Evaluation)
- **Клиентская ОС:** Windows 10 Pro
- **Среда виртуализации:** Oracle VirtualBox 7.0
- **Система управления доменом:** Active Directory Domain Services
- **Прикладное ПО:**
  - 1С:Предприятие 8.3 (учебная версия) – конфигурации «Бухгалтерия предприятия», «Управление небольшой фирмой»
  - Microsoft Office 365 (ознакомительная версия)
  - Kaspersky Endpoint Security (пробная версия)
- **Открытое ПО:** SQLite (сборка из исходников)
- **Система контроля версий:** Git / GitHub
